# WeatherAPI

## Author
Lokesh

## About the Project
WeatherAPI is a simple application that fetches and displays real-time weather information using a weather API.  
This project was developed to explore API integration, data handling, and user-friendly display of weather details.

## Features
- Real-time weather data
- Location-based search
- Clean and simple interface

## Purpose
This project was created as part of my academic work to understand API usage and practical application development.

## How to Run
1. Clone the repository
2. Install required dependencies
3. Run the application
4. Enter a location to get weather details
